Google Tasks for Pomotodo
===========================

[蕃茄土豆][^1] 是一个结合了蕃茄工作法和 GTD 的网页应用程序，是一个体验非常不错的效率工具。

**pomogtd** 是由一个游猴脚本演变而来，用于将蕃茄土豆中的**土豆**替换成 Google Tasks

![PomoGTD](http://pic.yupoo.com/greatghoul_v/CorL9BsI/wn7gx.png)
